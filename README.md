Esse é um projeto simples de Java que permite cadatrar, listar e buscar usuários no console.

Técnologias utilizadas:
- Java
- Programação Orientada a Objetos
- ArrayList

Funcionalidades:
- Cadastrar Clientes
- Listar clientes cadastrados
- Buscar cliente pelo nome
